## React and Redux Sagas Authentication App

This repo is an example application that documents and walks through a complete Signup, Login and Protected resource flow using the following:

- "react": "^16.2.0",
- "react-dom": "^16.2.0",-
- "react-redux": "^5.0.7",
- "react-router": "^4.2.0",
- "react-router-dom": "^4.2.2",
- "react-scripts": "1.1.1",
- "redux": "^3.7.2",
- "redux-form": "^7.3.0",
- "redux-saga": "^0.16.0",

This Project is based on the flow,Thanks for the author...

To start it up:

a) Git clone it

b) `yarn` or `npm i` to install dependencies

c) `yarn start`
